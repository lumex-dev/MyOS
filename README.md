# MyOS

MyOS is a personal productivity and task management application designed to help users organize tasks, projects, priorities, and schedules in a more intelligent and structured way.

The goal of the project is to build a system that reduces cognitive load by guiding users through task management and automatically providing relevant information at the right time.

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JavaScript

## Features

- User authentication
- Task management
- Project organization
- Priority and scheduling system
- Database persistence with PostgreSQL

## Installation

Clone the repository:
bash git clone git@github.com:lumex-dev/MyOS.git
Install dependencies:
bash npm install
Configure environment variables:
bash cp .env.example .env
Run database migrations:
bash npx prisma migrate dev
Start the development server:
bash npm run dev
